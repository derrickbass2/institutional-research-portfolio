# Institutional Research Portfolio (APA 7)

Reproducible Institutional Research portfolio using publicly available postsecondary datasets (e.g., IPEDS and state/system portals) with R-based analysis workflows and APA 7–formatted reporting.

## What’s inside
- **Projects**: Each project includes a short methods overview, reproducible scripts, and an APA 7 manuscript draft.
- **Data policy**: Raw datasets are not stored in this repository. Each project includes source links and retrieval instructions.

## Projects
1. **Retention & Completion (IPEDS, 2019–2023)** — multivariate modeling and diagnostics with transparent assumptions and limitations.

## Reproducibility
Each project folder contains:
- `analysis/requirements.R` (packages)
- `analysis/analysis.R` (main workflow)
- `paper/manuscript.md` + `paper/references.bib`
